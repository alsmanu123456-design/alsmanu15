# ⚠️ لا تحذف هذا الملف أبداً ⚠️

## هذا المجلد هو المجلد الأساسي للبوت

مجلد `alsmanu4/` هو **المجلد الرئيسي** الذي يحتوي على **كل** محتويات البوت.

---

## 📁 محتويات المجلد

| الملف/المجلد | الوصف |
|---|---|
| `dist/` | ملفات البوت المترجمة — **لا تحذفها أبداً** |
| `dist/index.mjs` | الملف الرئيسي للبوت (15 MB) |
| `wa-sessions/` | جلسات WhatsApp المحفوظة |
| `bot-data/` | بيانات المستخدمين والإعدادات |
| `config.json` | إعدادات البوت (GitHub Token) |
| `deploy.mjs` | ملف التشغيل الرئيسي |
| `startup.mjs` | ملف الإقلاع البديل |
| `github-sync.mjs` | رفع البوت لـ GitHub |
| `ai-test.mjs` | اختبار الذكاء الاصطناعي |
| `package.json` | مكتبات Node.js |

---

## 🔄 قاعدة التحديثات

> **أي تحديث للبوت يجب أن يكون داخل هذا المجلد دائماً**
> 
> لا تضع ملفات البوت خارج `alsmanu4/`

---

## 🚀 طريقة التشغيل

```bash
cd alsmanu4
node deploy.mjs
```

---

## 🔁 رفع البوت لـ GitHub

```bash
cd alsmanu4
node github-sync.mjs
```

---

## ⚠️ تحذيرات مهمة

- ❌ **لا تحذف** `dist/index.mjs` — البوت لن يعمل بدونه
- ❌ **لا تحذف** `wa-sessions/` — ستفقد جلسات WhatsApp
- ❌ **لا تحذف** `bot-data/` — ستفقد بيانات المستخدمين
- ❌ **لا تحذف** `config.json` — لن يعمل رفع GitHub
- ✅ **احتفظ دائماً** بنسخة احتياطية على GitHub
